# Change Log

Todas as mudanças notáveis do tema "Rust Industrial" serão documentadas neste arquivo.
O formato baseia-se em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/).

## [0.1.0] - 2026-04-26

### Adicionado

- Lançamento inicial do tema `Rust Industrial Dark` com paleta baseada em Carvão e Ferrugem.
- Lançamento inicial do tema `Rust Industrial Light` para contrastes em ambientes claros.
- **Realce Semântico Avançado (Rust):**
  - Variáveis mutáveis (`mut`) agora recebem sublinhado.
  - _Lifetimes_ (`'a`) destacados em rosa metálico e itálico.
  - _Macros_ (`println!`, `vec!`) em negrito e verde escuro/claro para contraste.
- Personalização de cores ANSI para o Terminal Integrado (Otimizado para o output do `cargo`).
- Suporte a colorização para arquivos `Cargo.toml` (Recomendado o uso com a extensão _Even Better TOML_).
- Bracket Pair Colorization (parênteses e chaves) em tons orgânicos e industriais.
